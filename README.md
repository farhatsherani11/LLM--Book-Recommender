#  AI-Powered Book Recommendation System

An intelligent book recommendation system that uses AI-powered semantic search and emotion-based filtering to recommend books based on user queries.

Built using LangChain, Hugging Face embeddings, ChromaDB, and Gradio.

---
#  Features

-  Semantic search using vector embeddings
-  Emotion-based filtering (Happy, Sad, Suspenseful, etc.)
-  Category filtering (Fiction / Nonfiction)
-  Interactive book gallery UI
-  Fast vector similarity search using ChromaDB
